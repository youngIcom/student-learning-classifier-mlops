# Bukti Alerting Grafana

Untuk target Advanced, tambahkan screenshot 3 alert rule dan notifikasi/test alert.

Alert rule yang disarankan:

1. API Down: `student_classifier_api_up == 0`
2. High Memory Usage: `student_classifier_memory_usage_percent > 80`
3. Model Artifact Missing: `student_classifier_model_artifact_ready == 0`

Nama file yang disarankan:

- `1.rules_api_down.png`
- `2.notifikasi_api_down.png`
- `3.rules_high_memory.png`
- `4.notifikasi_high_memory.png`
- `5.rules_model_artifact_missing.png`
- `6.notifikasi_model_artifact_missing.png`
