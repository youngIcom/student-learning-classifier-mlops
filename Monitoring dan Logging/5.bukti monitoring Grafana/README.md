# Bukti Monitoring Grafana

Dashboard Grafana wajib memuat username Dicoding `yesaya` pada nama dashboard.

Untuk target Advanced, sertakan minimal 10 panel/metrik berbeda. Query yang disarankan:

- `student_classifier_api_up`
- `student_classifier_model_artifact_ready`
- `student_classifier_cpu_usage_percent`
- `student_classifier_memory_usage_percent`
- `student_classifier_disk_usage_percent`
- `student_classifier_system_uptime_seconds`
- `process_resident_memory_bytes`
- `process_cpu_seconds_total`
- `process_open_fds`
- `python_gc_objects_collected_total`
- `prediction_requests_total`
- `prediction_errors_total`
- `prediction_latency_seconds_count`
- `prediction_class_total`
