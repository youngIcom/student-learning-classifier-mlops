# Bukti Monitoring Prometheus

Minimal sertakan screenshot Prometheus target `UP` dan query metrik. Metrik custom yang tersedia:

- `student_classifier_cpu_usage_percent`
- `student_classifier_memory_usage_percent`
- `student_classifier_disk_usage_percent`
- `student_classifier_api_up`
- `student_classifier_model_artifact_ready`
- `student_classifier_system_uptime_seconds`
- `student_classifier_system_boot_time_seconds`

Tambahkan screenshot query dari Prometheus bila belum ada.
