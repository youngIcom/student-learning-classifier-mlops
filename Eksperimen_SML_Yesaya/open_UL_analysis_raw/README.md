# Raw Dataset

Folder ini mendokumentasikan sumber raw dataset. Raw file OULAD disimpan lokal di:

```text
dataset/open+university+learning+analytics+dataset/
```

Dataset tidak disalin ke folder ini dan tidak di-commit ke Git karena `studentVle.csv` berukuran lebih dari 100 MB.

## Sumber

Kuzilek, J., Hlosta, M., & Zdrahal, Z. (2015). Open University Learning Analytics dataset. UCI Machine Learning Repository.

DOI: https://doi.org/10.24432/C5KK69
